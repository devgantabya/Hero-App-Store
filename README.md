**App Name:**
**Hero App Store**

<br/>

**Description:**
Hero App Store is a modern, user-friendly platform to discover, explore, and manage mobile and web applications all in one place. With a sleek interface and powerful search capabilities, users can browse trending apps, read detailed descriptions, check ratings and reviews, and install their favorite apps with just a click. Hero App Store also allows users to manage installed apps, track updates, and get recommendations tailored to their preferences, making app discovery fast, convenient, and enjoyable for everyone.

<br/>

**Technologies:**

- **Frontend:** React, React Router, Tailwind CSS
- **Backend:** Node.js
- **Data Storage:** LocalStorage (for installed apps), JSON data
- **Charts & Analytics:** Recharts for app ratings and download visualizations
- **Other Tools:** React Icons, Responsive UI, AJAX for dynamic search, toastify, etc.
